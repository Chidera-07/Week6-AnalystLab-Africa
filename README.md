# Apple (AAPL) Stock Market Analysis Using Python

## Project Overview

This project analyzes the historical stock performance of **Apple Inc. (AAPL)** using Python. The objective was to clean, transform, analyze, and visualize stock market data to identify trends, measure performance, and generate actionable insights through time-series analysis.

This was my **first Python data analytics project**, providing hands-on experience with data cleaning, feature engineering, exploratory data analysis (EDA), and financial time-series analysis.

---

## Objectives

* Clean and preprocess historical stock market data
* Perform time-series analysis
* Engineer meaningful analytical features
* Visualize stock price and trading trends
* Generate insights from historical market performance

---

## Dataset

* **Company:** Apple Inc. (AAPL)
* **Source:** Yahoo Finance
* **Period:** July 2021 – July 2026
* **Records:** 1,254 daily observations

### Original Features

* Date
* Open
* High
* Low
* Close
* Volume

---

## Technologies Used

* Python
* Pandas

## Data Cleaning & Preprocessing

The following steps were performed:

* Converted the Date column to datetime format
* Verified and corrected data types
* Rounded numerical values
* Sorted records chronologically
* Removed unnecessary columns
* Extracted Year, Month, Weekday, and Quarter

---

## Key Insights

* Apple's stock price increased by approximately **118.1%** during the analysis period.
* The closing price rose from **$145.38** to **$317.31**.
* Average daily trading volume was **65.04 million shares**.
* Daily returns averaged **0.08%**.
* Monthly returns averaged **1.53%**.
* Moving averages confirmed a strong long-term upward trend despite short-term market fluctuations.
